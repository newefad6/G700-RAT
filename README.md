## 📱 Overview

The **Android Security Assessment Framework** is a comprehensive proof-of-concept tool demonstrating various Android security vulnerabilities and attack vectors. It supports **Android 8.1 (API 24) through Android 16+ (API 36+)**.

**Purpose:** Help security researchers understand:
- How Android permissions can be abused
- Common security misconfigurations
- Potential attack surfaces
- Defense mechanisms against such tools
